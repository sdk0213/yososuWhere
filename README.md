# 요소수어디 
* 해당 앱에서 제공하는 데이터는 '공공데이터 포털 요소수 재고 확인' API 를 통하여 제공 받는 데이터로 **'공공데이터활용지원센터'** 에서 제공합니다.
---
### 기능
* 내 위치에서 가장 가까운 순서대로 목록에 출력
* 요소수 재고량 확인
---
### 업데이트 필요한 기능
* 맵 상에 마커로 요소수 재고량 뿌리기
* 검색 지원
